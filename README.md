# MaxwellMetzner.github.io

Personal portfolio site for Maxwell Metzner, deployed with GitHub Pages.

This is a static single-page web app built with HTML, CSS, and vanilla JavaScript. It highlights featured projects, professional background, and contact links, with a polished front-end focused on performance and presentation.

Current site features include:

- Responsive single-page portfolio layout
- Animated section reveals and smooth scrolling navigation
- Light and dark theme toggle with saved user preference
- Featured personal and school project sections
- SEO and social sharing metadata for GitHub Pages deployment

Project structure:

- `index.html` contains the page content and metadata
- `assets/css/style.css` contains the site styling and visual design
- `assets/js/script.js` contains client-side interactions and animations
